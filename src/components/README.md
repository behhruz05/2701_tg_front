# 📦 components — Qayta ishlatiladigan bo'laklar

Bu yerda **kichik va ko'p marta ishlatiladigan** komponentlar turadi.
Bitta sahifaga emas, **butun loyihaga** tegishli bo'laklar.

## ui/ — eng kichik "g'isht" komponentlar
Misol: `Button`, `Input`, `Avatar`, `Loader`, `Modal`
Bular hech narsani bilmaydi — faqat ko'rinish. Har joyda ishlatiladi.

## layout/ — sahifa "ramkasi"
Misol: `Sidebar` (chap menyu), `Header`, `ChatList`
Telegram'dagi doim turadigan chap panel va yuqori panel shu yerda.

> Qoida: agar komponent **2+ joyda** ishlatilsa — shu yerga qo'y.
