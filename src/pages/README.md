# 📄 pages — Sahifalar

Bu yerda **to'liq sahifalar** turadi. Har bir sahifa = bitta papka.

Foydalanuvchi ko'radigan asosiy ekranlar:

- `Login/`    → kirish / ro'yxatdan o'tish sahifasi
- `Chat/`     → asosiy chat ekrani (suhbatlar + xabarlar)
- `Settings/` → sozlamalar sahifasi (profil, mavzu va h.k.)

## Sahifa nima qiladi?
Sahifa o'zi ko'p kod yozmaydi. U faqat **bo'laklarni yig'adi**:
`components/` dan tugma/input oladi, `features/` dan funksiyani chaqiradi.

> Eslatma: sahifa = "boshqaruvchi", components = "qism".
