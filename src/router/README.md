# 🧭 router — Sahifalar yo'nalishi

Bu yerda **qaysi URL qaysi sahifani ochishi** belgilanadi.
`react-router-dom` kutubxonasi ishlatiladi.

Misol yo'nalishlar (keyin yozasan):
- `/login`    → Login sahifasi
- `/`         → Chat sahifasi (asosiy)
- `/settings` → Settings sahifasi

## "Himoyalangan" yo'nalish
Agar foydalanuvchi tizimga kirmagan bo'lsa — uni `/login` ga qaytarish.
Bu logika ham shu yerda bo'ladi (PrivateRoute).

> O'rnatish: npm install react-router-dom
