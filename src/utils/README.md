# 🛠 utils — Yordamchi funksiyalar

Bu yerda **kichik yordamchi funksiyalar** turadi. Reactga bog'liq emas —
oddiy JavaScript funksiyalar.

Misol:
- `formatTime`   → vaqtni chiroyli ko'rsatish ("14:30", "kecha")
- `validateEmail`→ email to'g'rimi tekshirish
- `truncate`     → uzun matnni qisqartirish ("Salom..." )

> Qoida: agar funksiya rasm/tugma chizmasa, faqat hisob-kitob qilsa — shu yerga.
