# 🎨 styles — Umumiy dizayn / ranglar

Bu yerda **butun loyihaga tegishli** CSS turadi.

Misol:
- `global.css`    → asosiy sozlamalar (font, body, reset)
- `variables.css` → ranglar (Telegram ko'k rangi, fon rangi va h.k.)

> Eslatma: bitta komponentga tegishli stil — o'sha komponent yonida tursin.
> Bu yerda faqat **hamma joyga** tegishli narsa.
